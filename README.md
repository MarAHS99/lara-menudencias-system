# 🧾 Lara Menudencias — Sistema de Ventas

Un sistema simple y rápido para gestionar ventas de menudencias sin depender de Excel.

Pensado para negocios reales que necesitan:
- rapidez
- claridad
- cero errores

---

## 🚀 ¿Qué hace?

Este sistema permite:

✅ Crear boletas de venta  
✅ Editar boletas existentes  
✅ Ver historial completo  
✅ Calcular totales automáticamente  
✅ Gestionar precios por producto  
✅ Controlar stock básico  

Todo directamente en el navegador.

---

## 💡 ¿Por qué existe?

Porque Excel se rompe.

- errores humanos
- fórmulas mal tocadas
- datos desordenados
- difícil de mantener

Este sistema reemplaza eso con un flujo claro y controlado.

---

## 🧠 Cómo funciona

- Todo se guarda en `localStorage`
- No necesita internet
- No necesita instalación
- Corre 100% en el navegador

---

## 📦 Estructura

- `ventas` → creación de boletas  
- `historial` → listado de ventas  
- `config` → precios de productos  
- `compras` → stock (en desarrollo)  
- `análisis` → métricas básicas  

---

## ⚙️ Funcionalidades clave

### 🧾 Boletas
- Cálculo automático por producto
- Total en tiempo real
- Saldo anterior + entrega → cálculo de deuda

### ✏️ Edición
- Click en historial → carga automática
- Edición segura sin pisar otras boletas

### 💰 Precios
- Configuración centralizada
- Autocompletado al cargar productos

---

## ⚠️ Limitaciones actuales

- Los datos se guardan en el navegador
- Si se borra el storage → se pierde la info
- No hay usuarios ni login
